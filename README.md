# Cafe-management-system
This a simple cafe management system using flask that handles clients logins, authentication and database.
It allows admin to add, edit and delete new products and staff.

Goal:
To learn and understand flask backend framework authenication, authorasation and database.
ensuring that admins and users are recognized and more secure logins.

What i Have learnt and implemented:
1. To work with flask routes
2. API integration
3. Authentication with flask_wtforms
4. Flask sessions
5. requests
6. Authorasation
7. password hashing using werkeug.security
8. Error handling
9. files upload
10. CRUD -->Create, Read, Update and Delete
11. Database model using sqlite3
12. Jinja2 on html docs
13. Templates inheritance
14. Flask logins
15. Post and Get
16. Sqlalchemy for database

Features:
1.Admin dashboard
2.Tables
3.Authenication pages
4.Add and edit and delete tables 
5.User's dashboards

Technologies:
